영업 설치 관리 시스템 solo

고객에게 영업을 한 내역을 관리하고, 영업 내역을 출력, 영업에 대한 설치 내역까지 모두 관리할 수 있는 웹사이트 및 안드로이드 앱입니다.

주요 기능

  • 사원 관리
  • 영업 및 설치 내역 관리
  • 영업 내역 출력
  • 설치 내역 엑셀 저장
  • 영업 내역 및 설치 내역 입력(앱)
  • 결제 관리 페이지

후기

 Jetpack Compose를 처음 사용했던 프로젝트입니다. Vue.js와 함께 Declarative UI 패러다임을 처음 적용하여 개발하였습니다. 수정사항이 많이 생겨서 힘들었던 기억이 있습니다.

Gecko solo

Gecko는 헬스장에서 트레이너가 미리 지정된 영상들을 통해 트레이너와 고객이 영상을 보며 운동을 할 수 있도록 도와주는 웹사이트입니다. 실시간 타이머 기능 또한 지원합니다.

주요 기능

  • 회원 및 지점 관리
  • 영상 업로드 및 영상 카테고리 관리
  • 업로드된 영상으로 플레이리스트 생성
  • 플레이리스트 재생(4,6분할 지원)
  • 실시간 타이머

후기

 기존에 사용했던 스택들을 사용하여 개발해서 빠르게 개발할 수 있었습니다. 실시간 타이머 기능을 개발할 때, polling 방식 또는 웹소켓 방식을 이용할 수 있었는데, 웹소켓 방식은 복잡성이 많이 생기는 반면, 약간의 반응성의 이득만 제공한다고 판단하여 polling 방식을 이용하였습니다.

Myopia Management solo

근시 관리 웹사이트로, 병원에서는 안구 성장 데이터를 측정하여 입력하면 reference 데이터와 함게 차트로 표시해주는 웹사이트입니다. 환자는 자신의 안구 성장 데이터와 차트를 환자 번호로 검색하여 확인할 수 있습니다.

주요 기능

  • 병원 등록
  • 병원 멤버 관리
  • 환자 등록
  • 안구 성장 데이터 입력
  • 환자 관리
  • 안구 성장 데이터 차트 표시

후기

 클라이언트가 요구사항을 자세하게 정의해주셨던 기억이 납니다. 그러나 문제점이 몇 군데 있어서 클라이언트에게 해결책을 제시하고, 의견을 수렴해 해결하였습니다. Sign in via Google 기능을 처음으로 구현했습니다.

미러톡톡 team

미러톡톡은 미용실 거울용 안드로이드 앱입니다. 안드로이드 앱이지만 대부분이 웹 뷰 기반으로 개발되어 있습니다.

개발한 기능

  • 웹뷰 위에 실시간 카메라 화면을 오버레이
  • 안드로이드 Room 데이터베이스 기반 Persistence Layer 개발
  • 웹뷰 인터페이스 개발
  • 안드로이드 네이티브 기능 개발
  • 웹뷰 기반 앱 기능 개발

후기

 제가 다니던 회사에서 타사에 외주를 준 프로젝트였습니다. 제가 참여했을 땐 회사쪽과 외주를 받은 회사 쪽의 커뮤니케이션이 원할하지 않아 원래 안드로이드 네이티브로 개발하려고 했으나, 웹뷰 기반으로 개발된 상태였습니다. 따라서 USB 기반 카메라 화면을 표시할 수 없는 등의 여러 문제가 발생하였습니다. 그래서 저의 역할은 안드로이드 네이티브 기능 개발과 웹뷰와 연동하는 인터페이스 개발이었습니다. 특히, USB 카메라 데이터를 웹뷰로 스트리밍 하기엔 무리가 있어서 오버레이 방식을 제안하였습니다. 또한 웹뷰 사이드의 javascript 개발에도 참여해, 공통 기능들을 개발하였습니다.
